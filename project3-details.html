Game Studio Management

GitHub

Details

Game Studio Management consists of two applications. The first is a server application programmed in Java that manages multiple clients simultaneously. It allows the management of clients and their interactions. The second is an Android client application programmed in Kotlin. his client application connects to the server and enables users to manage the purchase of games. Each registered user can access and manage a virtual wallet, modify and review their account information, purchase game sessions, or buy flat-rate subscriptions for games.
